# todolist

📌 투두리스트 웹 사이트


##[1] 기능

▫️ 해야할 일 등록 

▫️ 해야할 일 삭제

▫️ 해야할 일 수정

▫️ 해야할 일 완료

##[2] 개발 환경

IDE: IntelliJ IDEA

개발 언어: Java 21

빌드 도구: Gradle

프레임 워크: Spring Boot 3.4.4
